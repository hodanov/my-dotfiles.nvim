# 記事テンプレート

執筆時にこの骨格に沿って記事を構成する。
Front Matter は既存ファイルがある場合はそのまま保持し、`Title` のみ更新する。
新規作成の場合は以下の形式で始める。

## Front Matter

```markdown
---
Title: <記事タイトル>
Category:
  - <カテゴリ名>
Draft: true
---
```

## 記事本文の構成

```markdown
![サムネイル](./path/img.jpg)

[:contents]

## はじめに

<自己紹介と記事の概要。対象読者に何を伝えるかを簡潔に書く>

## <本題の見出し1>

<アウトラインに沿って展開する>

## <本題の見出し2>

<コード例や設定例を含め、具体的に書く>

## まとめ

<記事の要点を振り返り、読者へのメッセージで締める>
```

## 構成のポイント

- `![サムネイル]` と `[:contents]` は必ず記事冒頭に配置する
- Front MatterのTitleが h1 見出しのため、見出しレベルは `##`（h2）から始める
- コードブロックには言語指定を付ける
- 箇条書きと段落を使い分け、読みやすさを確保する
